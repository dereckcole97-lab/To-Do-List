# To-Do-List

## Description
Is simply a tool (or a checklist) built in HTML,CSS and JavaScript that helps to remember and organize what needs to be done. Created as one of my beginner's project to practice on functions and user input.

## Features
-Write down tasks
-Save the tasks as you go
-Set deadlines or reminders for the day
-Mark tasks as completed
-Possibility to reset

## Benefits
-Keeps you from forgetting things
-Helps you stay organized
-Reduces stress by clearing your mind
-Makes it easier to focus on what’s important
-Based on how they're written out, can make it easier to choose the important once

### Example
Rather than trying to remember everything that needs to be done for the day like meetings to set up or groceries or gym workout or some bills to be paid, this allows to have a visual clarity by typing them down.
